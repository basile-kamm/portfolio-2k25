Grenouche is a video created in my first year of apprenticeship. We had one week to work on a game with a small group, and we've chosen to make a video game. We've then developed the concept of a 2 player platformer, with the simple goal of catching the most flies in 90 seconds. My role in this project was to code all the game and making the animation sprites.

This work gave us great freedom and a lot of fun. It was the first time I coded in Unity, using the C# language, so I had to learn it all, but I really enjoyed working on a project with a playable result.

Project in collaboration with Anaïs Chassot, Lauriane Sauteur and Victor Bertschy
