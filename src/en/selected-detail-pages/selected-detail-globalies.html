Globalies is a website that aims to raise awareness about the importance of the ownership of the products we use every day. It aims to link the multinational companies to the products they sell, and to show the impact of their production on the environment and the people who make them.

This project was my first time using wordpress. I had to learn how to use it, and how to create a custom theme. I also had to learn how to use the ACF plugin, which allows to create custom fields in wordpress. I also had to create a poster, stickers and an instagram to showcase my project.
